## 브레인 스토밍

쉬운 문제. 서로의 배수라는 제약 조건이 있고, 오름차순으로 정렬도 되어있어서 O(n)으로 끊을 수 있었다. 좀 더 효율적으로 하기 위해 arr에 tot보다 큰 값은 안 받았다.



## 코드

```python
n, tot = map(int, input().split())
arr, res = [], 0

for _ in range(n):
    temp = int(input())
    if temp > tot: break
    arr.append(temp)
    
for i in range(len(arr)-1, -1, -1):
    res += tot//arr[i]
    tot %= arr[i]
    if not tot:
        print(res)
        break
```
