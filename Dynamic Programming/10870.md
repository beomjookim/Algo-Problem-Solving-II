## 브레인 스토밍

DP 메모이제이션 예시로 쓰이는 대표적인 문제. 재귀, DP, Bottom Up 의 세가지 방식 모두로 풀어보자.


## 코드

1. 재귀를 활용한 풀이
```python
def rec(n):
    if n <= 1: return n
    return rec(n-1) + rec(n-2)

print(rec(int(input())))
```

2. DP를 활용한 풀이
```python
n = int(input())

arr = [0, 1, 1]

def fib(n):
    if len(arr) <= n: arr.append(fib(n-1) + fib(n-2))
    return arr[n]

print(fib(n))
```

3. Bottom Up

```python
n = int(input())

arr = [0, 1, 1]
while len(arr) < n + 1: arr.append(arr[-1] + arr[-2])

print(arr[-1] if n >= 3 else arr[n])
```
