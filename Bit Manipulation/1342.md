## Number of Steps to Reduce a Number to Zero


### 브레인 스토밍

기본적인 비트 연산이 가능한지 물어보는 문제.


### 코드

```python
class Solution:
    def numberOfSteps (self, num: int) -> int:
        steps = (num == 0) - 1
        while num:
            steps += 1 + (num & 1)
            num >>= 1
        return steps
```
